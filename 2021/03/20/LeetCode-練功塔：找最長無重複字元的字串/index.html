LeetCode 練功塔：找最長無重複字元的字串

發表於 2021-03-20 更新於 2021-03-22

最近在 LeetCode 上寫題目練習分析問題和程式邏輯，我會把解出來的問題寫成文順便練習解釋。本篇問題是在給定字串內尋找最長無重複字元 (character) 的子字串 (substring)。

想自己玩玩看的可以點下方的題目連結。

LeetCode 的題目連結：https://leetcode.com/problems/longest-substring-without-repeating-characters/

問題

  • 要求：給你一個字串 s，要你找出裡面無重複字元的最長子字串的長度。
  • 傳入參數 s：內含有英文字母、數字、符號和空格，全部都是半形。格式是字串
  • 回傳：子字串長度。格式為數字。

分析

先大概想一下需要什麼。

  • 因為要確保無重複字元，需要比對
  • 需要紀錄目前為止最長子字串的字元數

比對流程：

  • 對象：尚未比對的字元和目前累積的無重複字元字串
  • 流程：
    1. 如果比對後未發生重複，將該字元加到目前累積字串的尾端。
    2. 如果發生重複，去除累積字串內重複字元的位置以前的子字串，包含該字串內的重複字元。
    3. 繼續比對。

字元數：因為累積字串會隨著比對結果改變，所以最長子字串的字元數目要在每次比對時更新。

如果在比對流程 2. 內才更新——只有在發生重複時才會更新——那會在字串 s 的字符全部都不一樣的時候出問題；因為沒有重複不會更新最長字元數。

壓縮資訊：

  • 目前累積的無重複字元字串：利用它是 s 的子字串這個特性，不必特地將它取出來存進另一個變數裡，只要掌握最前面和最後面兩個索引 (indices) 就能從 s 取出。

程式碼

第一版程式碼

  • 另外使用變數 substring 儲存目前累積的最常字串。
  • 因為參數字串 s 含有如倒斜線 \、括號 () 和雙引號 "" 等在正則表達式 (regular expression) 裡需要跳脫 (escape) 的字元，使用 String.search() 比對字串不太方便，所以用 String.split() 先換成 Array 後，再用 Array.find() 找衝突字元的位置。
  • 為使用 Array.forEach() 特地把 substring 用 String.split() 換成 Array。

自評：

  • For 迴圈也可以用來掃描字串的字元。相反的，為了用 Array 的內建函數特地轉成 Array 不僅占空間，而且這裡用到的內建函數結構上也是迴圈。多此一舉。
  • 變數 substring 是 s 的子字串，也就是說本來就存在 s 裡，只要用索引就能從 s 取出。

第二版程式碼

  • 用第一層 for 迴圈的第一欄位事先宣告兩個變數 start 和 end，標記目前累積最長字串的最前端和最末端索引，並用 end 表示掃描字串 s 的進度。
  • 第二層 for 迴圈則用於比對。比對位置從 start 到 end-1，碰到重複字元時更新 start位置至重複字元的下一個位置，並直接用 break 跳出。

自評：

  • 用 for 迴圈取代 Array 內部的方法簡化程式碼
  • 覺得用 break 有點彆扭，也許可以考慮一下更動終止條件或迴圈內部的操作。
