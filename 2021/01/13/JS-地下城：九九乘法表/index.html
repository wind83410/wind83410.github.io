JS 地下城：九九乘法表

發表於 2021-01-13 更新於 2021-01-24

本篇是系列文 JS 地下城的第一篇，用於記錄學習 Javascript 的過程。JS 地下城，是我在 Udemy 買的、網頁技術教學團隊六角學院所開的課程之一，用不同的題目協助修課學員練習 JS 語法。他們建議學員記錄每道題目解題過程和解釋自己的程式碼結構，故有此系列文。

閱讀前注意事項：

  • 以下會將 HTML 標籤和 DOM 元素分別略稱為「標籤」和「元素」，並且視兩者為同義詞交替使用。
  • 在此文未提及的語法大多是為了樣式或排版，本文會集中在學系重點的相關語法上，故與此較無關者僅做精簡解釋。

思考

在開始寫程式碼之前，先觀察一下設計圖上是否有可重複的單元（類似的排版等）。有的話可以先做那個單元，之後就能複製單元完成畫面；以本次題目而言，可重複的單元即是「單一的乘法表」，如下圖所示。

只要把左上角的「數字標題 2」和「乘法算式（比如 2 × 1 = 2）最前面的數字」替換成其他數字即可完成其他數字的乘法表。

程式碼

照上一段的思考，寫程式碼的流程為先產生重複單元，再以複製該單元完成畫面。

重複單元結構

接下來是大致安排重複單元的 DOM 結構。考慮結構的可讀性，大數字可以用標題標籤 <h2>，下方的乘法算式可用無序列表 <ul> 和 <li> 的組合來做；最高層級的標題標籤 <h1> 則留給整個頁面的標題「九九乘法表」。

其中 i 為 2 到 9 的數字，而 i 的倍數會在產生時換成相對應的數字。大概想好結構之後，就能開始寫 Javascript 產生重複單元。

生成重複單元

這裡採取的作法，主要使用兩種函數：document.createElement() 和 Node.appendChild()；前者負責產生新的元素，後者則負責把元素 A 加到另一個元素 B 讓 A 變成 B 的子元素，一次只能加一個。（後者亦可用新的 API ParentNode.append() 取代，可同時加多個元素）

首先，先產生重複單元最外層的 <div>。document.createElement() 此函數接受的參數為標籤名稱字串，所以只要在括號裡放進 'div' 即可產生。產生後以常數格式 const 儲存。用同樣的方式產生標題 <h2> 和空的無序列表 <ul>。

此時標題還是空白，用元素的 textContent 屬性加入數字到標籤內。

這裡 i 還沒有在程式內定義，它是代表乘法表的數字標題，等一下複製單元時會用到。

複製並加進 DOM 裡

再來只要複製單元加進 DOM 就能完成畫面。這裡因為要一次完成 2 到 9 的乘法表，所以一樣用 for 迴圈處理，前文所述的未定義變數 i 就是這時候用上。
